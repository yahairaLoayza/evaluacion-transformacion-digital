# Gestión de Prospectos Comerciales – Odoo

## Objetivo
Configurar un flujo básico y funcional para la gestión de prospectos comerciales utilizando Odoo Online.

## Pasos realizados
1. Creación de un usuario con permisos básicos de CRM y Contactos.
2. Uso del módulo CRM para la gestión de leads.
3. Registro de prospectos con campos: nombre, correo electrónico y teléfono.
4. Seguimiento del estado del prospecto mediante la vista Kanban.

## Módulos utilizados
- CRM
- Contactos

## Cómo replicar la configuración
1. Crear una cuenta en Odoo Online.
2. Activar el módulo CRM.
3. Crear un usuario con permisos básicos.
4. Registrar prospectos y actualizar su estado.
